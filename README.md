# Medicare AI Advisor

AI-powered Medicare Advantage plan comparison and enrollment platform. Chat-first conversational experience with deep CMS data integration.

## Features

- **Conversational AI Interface** - Chat naturally about your Medicare needs
- **Plan Comparison** - Side-by-side comparison of every MA plan in your ZIP
- **Drug Coverage Check** - Verify your prescriptions are covered
- **Provider Search** - Confirm your doctors are in-network
- **Digital Enrollment** - Complete enrollment 100% online
- **CMS Star Ratings** - Official quality ratings for every plan

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **AI**: Perplexity Sonar API
- **Icons**: Lucide React
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

```
PERPLEXITY_API_KEY=your_api_key_here
```

## Data Sources

- CMS Medicare Plan Finder (medicare.gov)
- CMS Quality Star Ratings
- Plan sponsor formulary data
- Provider network directories
